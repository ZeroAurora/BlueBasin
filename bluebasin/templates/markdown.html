Markdown 使用指南

什么是 Markdown

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。其基本语法可以在 这里 查看。

本网站中的 Markdown

本网站使用 markdown-it-py 实现 Markdown 的渲染，兼容 CommonMark 规范。高亮使用 Prism。

限制

语法限制

  • 内容中使用的 HTML 语法会被直接渲染（称为 escaping）。

  • 为确保页面排版整洁，请不要使用 h1（#）与 h2（##），并从 h3（###）开始排列标题。

  • 请使用两个换行来开始新的段落。

长度限制

本网站目前单条内容（含 Markdown 语法）的长度限制为 10000 字符，若超出则会无法发布。请考虑将长文分段发布。
